# 🍕 Pizza Sales Analysis

## Project Overview
Analysis of a pizza restaurant's full year of sales data (Jan–Dec 2015) 
to uncover revenue trends, peak trading periods, category performance, 
and menu optimisation opportunities.

## Tools Used
- **SQL** — data extraction and querying
- **Microsoft Excel** — data cleaning, KPI calculation and dashboard
- **Power BI** — interactive dashboard and visualisation

## Key KPIs
| Metric | Value |
|---|---|
| Total Revenue | $817,860 |
| Total Orders | 21,350 |
| Total Pizzas Sold | 49,574 |
| Average Order Value | $38.31 |
| Average Pizzas Per Order | 2.32 |

## Key Insights
- **Classic category** led all sales at 26.91% — 14,888 pizzas sold
- **Large size** pizzas drove 45.89% of all sales volume
- **Peak hours** were 12–1 PM and 4–8 PM daily
- **Fridays & Saturdays** were the busiest days consistently
- **July & January** recorded the highest orders of the year
- **Thai Chicken Pizza** generated the highest revenue (~$43K)
- **Brie Carre Pizza** was the worst performer across revenue, quantity and orders
